# WebFlosy - Premium Website Builder Service

A high-quality website template for a web agency, built with Next.js and premium dark mode aesthetics.

## Features
- **Modern Design**: Glassmorphism, Gradient Text, Smooth Transitions.
- **Comprehensive Structure**: 
  - **Service**: Detailed process and features.
  - **Portfolio**: Filterable grid and case study layout.
  - **Pricing**: Clear packages and timeline visualization.
  - **FAQ**: Accordion style questions.
  - **Contact**: Integrated form and schedule availability.

## Tech Stack
- **Framework**: Next.js 14 (App Router)
- **Styling**: Native CSS Modules (No Tailwind dependency) & CSS Variables
- **Icons**: Lucide React
- **Language**: TypeScript

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) with your browser.

## customization
- Edit `src/app/globals.css` to change color themes (CSS Variables).
- Modify `src/styles/*.module.css` for component specific styles.
